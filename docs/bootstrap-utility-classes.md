Here's a comprehensive list of Bootstrap utility classes and their CSS equivalents:

## Display
- `d-none` → `display: none;`
- `d-block` → `display: block;`
- `d-inline` → `display: inline;`
- `d-inline-block` → `display: inline-block;`
- `d-flex` → `display: flex;`
- `d-grid` → `display: grid;`

## Flexbox
- `flex-row` → `flex-direction: row;`
- `flex-column` → `flex-direction: column;`
- `justify-content-start` → `justify-content: flex-start;`
- `justify-content-center` → `justify-content: center;`
- `justify-content-end` → `justify-content: flex-end;`
- `justify-content-between` → `justify-content: space-between;`
- `align-items-start` → `align-items: flex-start;`
- `align-items-center` → `align-items: center;`
- `align-items-end` → `align-items: flex-end;`
- `flex-wrap` → `flex-wrap: wrap;`

## Spacing (m = margin, p = padding)
- `m-0` → `margin: 0;`
- `mt-3` → `margin-top: 1rem;`
- `mb-3` → `margin-bottom: 1rem;`
- `ms-3` → `margin-left: 1rem;` (start)
- `me-3` → `margin-right: 1rem;` (end)
- `mx-3` → `margin-left: 1rem; margin-right: 1rem;`
- `my-3` → `margin-top: 1rem; margin-bottom: 1rem;`
- `p-3` → `padding: 1rem;`
- `pt-3` → `padding-top: 1rem;`
- `pb-3` → `padding-bottom: 1rem;`
- `px-3` → `padding-left: 1rem; padding-right: 1rem;`
- `py-3` → `padding-top: 1rem; padding-bottom: 1rem;`

Spacing scale: 0=0, 1=0.25rem, 2=0.5rem, 3=1rem, 4=1.5rem, 5=3rem

## Gap
- `gap-1` → `gap: 0.25rem;`
- `gap-2` → `gap: 0.5rem;`
- `gap-3` → `gap: 1rem;`

## Text Alignment
- `text-start` → `text-align: left;`
- `text-center` → `text-align: center;`
- `text-end` → `text-align: right;`

## Text Formatting
- `fw-bold` → `font-weight: bold;`
- `fw-normal` → `font-weight: normal;`
- `fst-italic` → `font-style: italic;`
- `text-decoration-none` → `text-decoration: none;`
- `text-uppercase` → `text-transform: uppercase;`
- `text-lowercase` → `text-transform: lowercase;`

## Colors

### Text Colors
- `text-primary` → `color: #0d6efd;` (blue)
- `text-secondary` → `color: #6c757d;` (gray)
- `text-success` → `color: #198754;` (green)
- `text-danger` → `color: #dc3545;` (red)
- `text-warning` → `color: #ffc107;` (yellow)
- `text-info` → `color: #0dcaf0;` (cyan)
- `text-light` → `color: #f8f9fa;` (light gray)
- `text-dark` → `color: #212529;` (dark gray)
- `text-white` → `color: #ffffff;`
- `text-black` → `color: #000000;`
- `text-muted` → `color: #6c757d;` (muted gray)

### Background Colors

- `bg-secondary` → `background-color: #6c757d;` (gray)
- `bg-success` → `background-color: #198754;` (green)
- `bg-danger` → `background-color: #dc3545;` (red)

- `bg-light` → `background-color: #f8f9fa;` (light gray)
- `bg-dark` → `background-color: #212529;` (dark gray)
- `bg-white` → `background-color: #ffffff;`
- `bg-black` → `background-color: #000000;`
- `bg-transparent` → `background-color: transparent;`

### Border Colors
- `border-primary` → `border-color: #0d6efd;`
- `border-secondary` → `border-color: #6c757d;`
- `border-success` → `border-color: #198754;`
- `border-danger` → `border-color: #dc3545;`

- `border-light` → `border-color: #f8f9fa;`

## Width/Height
- `w-25` → `width: 25%;`
- `w-50` → `width: 50%;`
- `w-75` → `width: 75%;`
- `w-100` → `width: 100%;`
- `h-100` → `height: 100%;`

## Position
- `position-relative` → `position: relative;`
- `position-absolute` → `position: absolute;`
- `position-fixed` → `position: fixed;`
- `top-0` → `top: 0;`
- `bottom-0` → `bottom: 0;`
- `start-0` → `left: 0;`
- `end-0` → `right: 0;`

## Border
- `border` → `border: 1px solid #dee2e6;`
- `border-0` → `border: 0;`
- `rounded` → `border-radius: 0.25rem;`
- `rounded-circle` → `border-radius: 50%;`

## Visibility
- `visible` → `visibility: visible;`
- `invisible` → `visibility: hidden;`

## Overflow
- `overflow-hidden` → `overflow: hidden;`
- `overflow-auto` → `overflow: auto;`
